# Luxury Watch Product Landing Page

A premium, minimalist product landing page for luxury watches built with React and Vite.

## Features

- **Modern Design**: Clean, minimalist aesthetic with generous whitespace
- **Premium Typography**: Uses Inter and Playfair Display fonts for elegance
- **Interactive Elements**: 
  - Color selection with visual feedback
  - Quantity controls
  - Add to cart functionality
- **Responsive Layout**: Fully responsive across all device sizes
- **Performance**: Optimized with Vite for fast loading

## Design Highlights

### Color Palette
- Primary: #1a1a1a (Dark Charcoal)
- Secondary: #666666 (Medium Gray)
- Accent: #b76e79 (Rose Gold)
- Background: #f8f8f8 (Light Gray)
- White: #ffffff

### Typography
- **Primary Font**: Inter (sans-serif) - for body text and UI elements
- **Secondary Font**: Playfair Display (serif) - for headings and brand elements

### Spacing System
- Extensive use of whitespace for premium feel
- Consistent spacing scale (0.5rem, 1rem, 2rem, 4rem, 8rem)

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or pnpm

### Installation
```bash
cd react-test
npm install
```

### Development
```bash
npm run dev
```
Open http://localhost:5173 in your browser.

### Build for Production
```bash
npm run build
```

## Interactive Features

1. **Color Selection**: Click on any color swatch to change the watch display
2. **Quantity Control**: Use +/- buttons to adjust quantity
3. **Add to Cart**: Click the main CTA button to simulate adding to cart
4. **Responsive Navigation**: Navigation adapts to screen size

## Performance Optimizations

## Design Principles

1. **Minimalism**: Less is more - focus on essential elements
2. **Whitespace**: Generous spacing for readability and elegance
3. **Typography**: Hierarchical font system for clear information architecture
4. **Consistency**: Uniform spacing, colors, and interactions
5. **Accessibility**: Semantic HTML and proper contrast ratios

The design embodies luxury through restraint, using space and typography to create a premium experience that lets the product speak for itself.
